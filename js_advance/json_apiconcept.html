XMLHttpRequest:(Legacy Method)

XMLHttpRequest (XHR) is a JavaScript object that allows a browser to send and receive data from a server without reloading the page.
[EventTarget <---- XMLHttpRequestEventTarget <---- XMLHttpRequest]
[readyState → request ka status (0 to 4)] [status → HTTP status (200 = success)] [responseText → server se aaya data]

Value State Meaning
0 UNSENT Client has been created. open() not called yet.
1 OPEND open() has been called.
2 HEADERS_RECIEVED send() has been called, and headers and status are available.(request received)
3 LOADING Downloading: responseText holds partial data.(processing)
4 DONE The operation is complete.(response ready)
